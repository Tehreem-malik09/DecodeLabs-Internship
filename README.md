Password Strength Checker

 Overview
This project is developed as part of the DecodeLabs Cyber Security Internship Program.

The Password Strength Checker evaluates a user's password and classifies it as Weak, Medium, or Strong based on security criteria.

 Features
- Checks password length
- Detects uppercase letters
- Detects numeric digits
- Detects special characters
- Provides password strength rating

 Technologies Used
- Python 3

Project Structure

└── main.py

 How to Run

